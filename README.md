# StudyMate
This project is an AI-powered PDF Learning Assistant designed to transform static documents into interactive learning experiences.
It enables users to:

Upload and process PDFs locally or with cloud AI support.

Extract and summarize key information.

Ask context-based questions and receive accurate answers.

Generate adaptive quizzes to reinforce learning.

Track progress through analytics and visual reports.

By combining local processing for privacy and cloud-based AI for advanced reasoning, the system offers a fast, secure, and personalized way to study and understand complex materials.
